##Recipe Project

**Course: / Project type:** Foundations in Odin Project / Simple Website

---

##Description

This project is a simple recipe website created for recipe project under HTML Foundations in Foundations course. This is created only using HTML. It demonstrates the use of basic HTML elements such as headings, paragraphs, lists, images and links.

---

##Features
- Recipe details with ingredients and preparation steps  
- Images of finished dishes  
- Lists for ingredients and steps
